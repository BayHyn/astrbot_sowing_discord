让 AI 学会搬史, 搬史进入新时代! 全自动搬史!

## ✨ 功能

- 🚫 自动从指定的群聊(推荐加入 1032915502, 共建搬史中心), 转发聊天记录(史)到其他指定的群聊(详见设置)

## 🛠️ 使用方法

1. 加入天天发史的群聊
2. 设置该群聊为搬史源头, 设置目标群聊
3. bot 开始全自动搬史!
